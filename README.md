# Organizador de listas em Python

## Sobre o projeto

Este projeto foi desenvolvido como exercício prático de Python, com foco em manipulação de listas, entrada de dados, estruturas condicionais e laços de repetição.

A proposta do programa é permitir que o usuário digite diferentes números inteiros e, ao final, exibir uma lista completa com todos os valores informados, além de duas listas específicas: uma apenas com números pares e outra apenas com números ímpares.

## Objetivo do exercício

O objetivo principal é praticar a organização de dados em listas a partir de uma condição lógica. Neste caso, o programa avalia cada número digitado pelo usuário e classifica o valor conforme o seu resultado na divisão por 2.

Com isso, o exercício ajuda a reforçar conceitos importantes da linguagem Python, como criação de listas, uso do `append()`, estrutura `while`, validação com `if` e `else`, além do operador módulo `%`.

## Funcionalidades

- Recebe números inteiros digitados pelo usuário
- Armazena todos os valores em uma lista principal
- Identifica quais números são pares
- Identifica quais números são ímpares
- Organiza os valores em listas separadas
- Permite continuar ou encerrar a entrada de dados
- Exibe o resultado final de forma simples e organizada

## Tecnologias utilizadas

- Python 3

## Como executar o projeto

1. Certifique-se de que o Python esteja instalado no computador.
2. Baixe o arquivo organizador-de-listas.py do projeto.
3. Execute o arquivo pelo terminal ou por uma IDE de sua preferência, como:
   - PyCharm
   - Visual Studio Code
   - IDLE

## Exemplo de Uso

Entrada:
- Digite um valor: 4
- Deseja continuar? [S/N] S
- Digite um valor: 7
- Deseja continuar? [S/N] S
- Digite um valor: 10
- Deseja continuar? [S/N] N

Saída:
- A lista completa é [4, 7, 10]
- A lista de pares é [4, 10]
- A lista de ímpares é [7]
